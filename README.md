# spy-logistic-regression-classification
Binary classification of SPY market direction using Logistic Regression and feature engineering in Python.
## Project Overview

This project predicts the daily direction of the SPY ETF using a Logistic Regression classification model.

Main steps:
- Data preprocessing
- Feature engineering
- Model training
- Model evaluation

Tools used:
- Python
- Pandas
- Scikit-learn
- Matplotlib
